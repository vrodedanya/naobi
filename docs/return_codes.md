# Return codes

## **(10-19) - Module importing error**
- ### 10 - Failed to open file
- ### 11 - Import itself
## (20-29) - Reserved
## **(30-39) - Unit compile error**
- ### 30 - Unit already exist
## (40-49) - Reserved
## **(50-59) - Code compile error**
- ### 50 - Unknown line
- ### 51 - Is already exist
- ### 52 - Doesn't exist
- ### 53 - Type error
- ### 54 - Wrong format
- ### 55 - Unknown operator
- ### 56 - Incorrect brackets
- ### 57 - Invalid arguments
## (60-69) - Reserved
## (70-79) - Reserved
## (80-89) - Reserved
## **(90-99) - Runtime errors**
## (100-109) - Reserved
## **(110-119) - Other**
- ### 110 - Not specified
- ### 111 - Keyword used as the name
- ### 112 - Internal error
